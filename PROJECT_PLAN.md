# 粒径计数台：项目方案

## 目标

将已经验证过的清洁度图片统计流程做成一套本地软件，使检测人员能够独立完成图片导入、比例校准、统计区域确认、分档计数和结果导出。

## 当前版本（MVP）

1. 导入 JPG、PNG、TIFF 或 BMP 原图。
2. 自动识别右下角黄色比例尺两条竖线的中心距，也可手工输入。
3. 在图片上调整椭圆统计区域，圆边自动向内留出保护带。
4. 使用深色核心＋边缘恢复的双阈值方法分割颗粒。
5. 以最大 Feret 径分为四档：25–50、50–100、100–200、＞200 μm。
6. 生成彩色标注图、汇总 CSV、逐颗粒明细和 ZIP 结果包。
7. 全程在本机运行，不上传图片。
8. 保存并快速切换不同相机、滤膜对应的参数模板。

## 技术结构

```text
浏览器工作台
  ├─ 图片与区域预览
  ├─ 比例尺和识别参数
  └─ 结果及文件下载
          │ 本机 HTTP
Python 本地服务
  ├─ 上传与任务管理
  ├─ OpenCV 颗粒识别引擎
  ├─ 标尺自动检测
  └─ CSV / JPG / ZIP 导出
```

运行时仅依赖 Python、NumPy 和 OpenCV，不依赖云服务或数据库。

## 识别口径

- 比例尺：两条黄色竖线的中心距对应标注长度。
- 统计尺寸：颗粒最大长度（最大 Feret 径）。
- 统计区域：用户确认的椭圆边界向内收缩指定距离。
- 颗粒判断：高阈值确认深色核心，低阈值恢复完整边缘，减少滤膜纹理误检。

## 后续版本
