# 粒径计数台：项目方案

## 目标

将已经验证过的清洁度图片统计流程做成一套本地软件，使检测人员能够独立完成图片导入、比例校准、统计区域确认、分档计数和结果导出。

## 当前版本（1.1.0 开发分支）

1. 导入 JPG、PNG、TIFF 或 BMP 原图。
2. 自动识别右下角黄色比例尺两条竖线的最外侧边缘距离，也可手工输入。
3. 在图片上调整椭圆统计区域，圆边自动向内留出保护带。
4. 使用深色核心＋边缘恢复的双阈值方法分割颗粒。
5. 以最大 Feret 径统计颗粒尺寸，提供相互独立的自定义模式和 VDA 19.1 模式（阶段 1）。
6. 生成彩色标注图、汇总 CSV、逐颗粒明细和 ZIP 结果包。
7. 全程在本机运行，不上传图片。
8. 保存并快速切换不同相机、滤膜对应的参数模板。
9. 使用黄金数据集验证精确率、召回率、分档、比例尺和重复性。
10. 人工新增、删除、拆分与撤销颗粒修正，并记录审计轨迹。
11. 输出带样品信息、原图摘要和算法版本的 PDF 报告。
12. 批量处理文件夹，并支持构建无终端窗口的 macOS 和 Windows 应用。
13. VDA 19.1 模式使用 E–N 尺寸等级和 `[下限, 上限)` 边界，50 μm 至少覆盖 10 像素，否则停止分析。
14. VDA 模式禁止通过手工输入尺寸新增或拆分颗粒，仅允许删除误检和撤销。
15. 分析结果明确记录当前 VDA 模式属于部分标准化实现，不构成完整合规声明。

## 技术结构

```text
浏览器工作台
  ├─ 图片与区域预览
  ├─ 比例尺和识别参数
  └─ 结果及文件下载
          │ 本机 HTTP
Python 本地服务
  ├─ 上传与任务管理          (app.py)
  ├─ 分析模式与尺寸等级      (analysis_modes.py)
  ├─ OpenCV 颗粒识别引擎     (engine.py)
  ├─ 人工复核与审计          (review.py)
  ├─ PDF 报告                (reporting.py)
  └─ 批处理                  (batch.py)
```

engine.py 模块拆分：

| 函数 | 职责 |
|------|------|
| `_read_and_normalize` | 图片读取 + 16-bit/灰度图归一化 |
| `detect_yellow_scale_gap` | 黄色比例尺双竖线检测，返回最外侧边缘距离及追溯坐标 |
| `_make_ellipse_mask` | 椭圆 ROI 蒙版生成 |
| `_runs` / `_bin_index` / `_maximum_feret_diameter` | 工具函数（连续区间、分桶、凸包直径） |
| `analyze_image` | 主分析管道，按分析模式执行分档和分辨率检查 |
| `_write_result_files` | 结果文件写入（图片/CSV/JSON/ZIP） |

analysis_modes.py 将模式差异集中在一个模块中：

| 模式 | 最小报告尺寸 | 分档边界 | 尺寸等级 | 分辨率门槛 |
|------|--------------|----------|----------|------------|
| 自定义模式 | 25 μm（不含） | `(下限, 上限]` | 25–50、50–100、100–200、＞200 μm | 不强制 |
| VDA 19.1 阶段 1 | 50 μm（包含） | `[下限, 上限)` | E–N | 50 μm ≥ 10 px |

模块级可调参数（`_*` 前缀常量，共 27 个）：

- 黄色检测：`_YELLOW_HSV_LOWER/UPPER`、`_YELLOW_MIN_AREA` 等 11 个
- 分析参数：`_GAUSSIAN_SIGMA_*`、`_BKG_GRAY_MAX`、`_JPEG_QUALITY_*` 等 16 个

运行时依赖 Python、NumPy、OpenCV 和 Pillow，不依赖云服务或数据库。

## 识别口径

- 比例尺：两条黄色竖线最外侧边缘的距离对应标注长度；自动检测会在 `analysis.json` 中保存外沿和中心坐标。
- 统计尺寸：颗粒最大长度（最大 Feret 径）。
- 统计区域：用户确认的椭圆边界向内收缩指定距离。
- 颗粒判断：高阈值确认深色核心，低阈值恢复完整边缘，减少滤膜纹理误检。
- VDA 阶段 1：仅完成标准尺寸等级、边界规则和分辨率门槛。纤维识别、标准分割配置、边缘颗粒重构与完整计量追溯仍待实现。

## 后续版本

- 根据实际检测规程扩展报告签字栏和清洁度编码。
- 增加 Feret min、最大内切圆、骨架伸展长度和纤维识别。
- 增加基于真实轮廓的人工新增与拆分，并在修正后重新测量。
- 建立 VDA 标准分割配置、边缘颗粒处理和可追溯校准记录。
